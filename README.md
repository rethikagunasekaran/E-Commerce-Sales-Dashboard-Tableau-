# E-Commerce Sales Dashboard (Tableau)

## Project Overview
This project focuses on analyzing an e-commerce dataset to understand sales performance, customer behavior, and product return patterns. The goal was to transform raw order data into an interactive dashboard that tracks key business metrics and highlights areas that require attention.

The dashboard provides a high-level view of sales, profit, discounts, customer segments, regional performance, and return reasons. It is designed to support quick decision-making by presenting important metrics in a clear and structured format.

## Tools Used
- **Tableau** (for data visualization and dashboard creation)
- **Excel/CSV dataset** (e-commerce orders, customers, and returns)

---

## Data Preparation
The dataset contained order-level information including sales, profit, discount, region, customer segment, and return details.

During the preparation stage:
- Checked for missing or inconsistent values.
- Verified the relationship between order and return datasets.
- Created calculated fields for:
  - Total Sales
  - Total Profit
  - Discount Percentage
  - Return Rate

---

## Key Dashboard Metrics
The dashboard includes the following key indicators:

- **Total Sales:** Overall revenue generated.
- **Total Profit:** Net profit from all orders.
- **Discount Percentage:** Average discount applied on sales.
- **Return Rate:** Percentage of orders that were returned.

---

## Visualizations Included

### Region vs Sales
- Compares sales performance across regions.
- Helps identify high- and low-performing areas.

### Category vs Sales
- Shows revenue contribution from each product category.

### Segment vs Profit
- Compares profit generated by different customer segments.

### Top 10 Customers
- Highlights customers who contribute the most to total sales.

### Monthly Sales Trend
- Displays sales performance over time.
- Helps identify seasonal patterns.

### Sales by State (Map)
- Provides a geographical view of revenue distribution.

### Return Analysis
- Shows the main reasons for product returns.
- Helps identify operational or quality issues.

### Segment vs Return Reason
- Compares return causes across customer segments.

---

## Key Insights
- The **West region** recorded the highest sales.
- The **Furniture category** contributed the most revenue.
- The **Corporate segment** generated the highest profit.
- A small group of top customers contributed a significant share of total sales.
- **Damaged products and late deliveries** were among the main return reasons.

---

## Business Recommendations
Based on the analysis:

- Focus marketing efforts on high-performing regions and segments.
- Review cost and pricing strategies in lower-profit categories.
- Investigate the main return reasons to reduce product returns.
- Use monthly trends for better inventory and promotion planning.

---

## Conclusion
This dashboard converts raw e-commerce data into clear and actionable insights. It provides a clear overview of sales performance, customer trends, and return patterns. The interactive filters allow users to explore the data by region, segment, and category, making it useful for both operational and strategic decisions.
